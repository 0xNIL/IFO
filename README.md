# 0xNIL factory

Contracts for the NIL token and the related free distribution

More info at [0xNIL.org](http://0xNIL.org)

### Testing

Install `testrpc` and `truffle`:

```
npm install -g testrpc truffle

```

Run `testrpc` generating 20 accounts:
```
testrpc -a 20
```

Run the tests:
```
truffle test

```
