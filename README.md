# 0xNIL Token

Contracts for the NIL token and the related IFO

More info at [0xNIL.org](http://0xNIL.org)

### Testing

Install `testrpc` and `truffle`:

```
npm install -g testrpc truffle
```

Install the dependencies
```
npm install
```

Run `testrpc` generating 20 accounts:
```
testrpc -a 20
```

Run the tests:
```
truffle test

```
